## Features
- Difficulty levels: Easy, Medium, Hard.  
- **New Game**, **Hint**, **Solve**, and **Print PDF** buttons.  
- Timer, Coins system, and Highscore saved using LocalStorage.  
- Generate Challenge ID to compete with friends.
